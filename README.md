## Covid-19-KnowledgeGraph
#这是知识工程大作业的源码。
数据集来源：http://openkg.cn/dataset/covid-19-baike中的COVID-19-wiki.JSON
本项目使用py2neo来操作图数据集neo4j进行知识图谱构建和搜索应用。
其中build_graph.py用于知识图谱构建，search_graph.py是基于命令行的知识图谱搜索应用
